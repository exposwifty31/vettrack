VetTrack is **operational software for active medical shifts**. The visual vocabulary is clinical-warm: deep forest-green chrome over warm parchment surfaces, lucide icons throughout, Plus Jakarta Sans + Heebo type, no emoji, no decorative gradients. Hebrew RTL is the default product locale.
