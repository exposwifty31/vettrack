05 · Elevation & materials

A five-level depth ladder

Design review flagged that surfaces read as background → card → card → card — too little separation between a resting card and something actually interrupting the user. Shadow now grows with stakes, not decoration: --shadow-card stays barely-there, while panels, modals and emergency overlays each step up in blur, spread and opacity. The real app maps onto this ladder 1:1 — Phase 10 in design-system-updates/.

Decisions & flags

Resolved

  • §13.2 Brand — indigo #4F46E5 / #818CF8 (shipped mark), --brand = --primary; navy-indigo hero ink; green #15803D reserved for scan/action.
  • §13.1 Scan — unified to /scan.
  • §13.6 Numerals — DM Mono, tabular figures.
  • Model — Apple grouped neutrals + system status colors mapped to clinical meanings.
  • Elevation — design review flagged flat depth hierarchy; adopted a 5-level ladder (background → card → panel → modal → emergency). Real app aligned to match in design-system-updates Phase 10 (§29).
  • Surface system — ivory ramp grows from 2 steps to 5 (bg/surface/panel/hover/active); real app aligned in Phase 11 (§30). Brand-color half of the same review is not yet resolved — real primary is indigo (matches the logo), not the forest green the review assumed; asked rather than guessed.
  • Phase 21 batch — 15-item polish review (section headers, card rhythm, soft surfaces, stat cards, status dots, tables, empty states, search inputs, context chips, staff/room cards, clinical timeline). Several items were already substantially shipped (accent rails = the criticality system below, soft surfaces = the shadow-less card above); rest verified against real source and built for real.
